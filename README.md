## Prerequisites
- Node 18
- yarn

## MacOS

Do the following from the root project directory:

1. Install dependencies
### `sudo yarn install`

2. Build component locally
### `yarn build`

3. Start local server and navigate to the URL listed in the terminal window to run the application
### `sudo yarn start`
